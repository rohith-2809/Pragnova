# Pragnova Technologies

Engineering the future through scalable, secure, and aesthetically powerful AI-integrated platforms and advanced digital solutions.

## 🚀 Overview

Pragnova Technologies is a next-generation tech solutions company focused on building intelligent digital systems, high-performance web applications, and AI-driven products. We unify strategy, design, development, and AI into a single, synchronized momentum.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS
- **Animations:** Framer Motion, GSAP
- **Icons:** Lucide React
- **Typography:** Inter (Google Fonts)
- **Smooth Scrolling:** Lenis

## 🏗️ Design System

Our design system is built on a "Deep Sea" obsidian aesthetic, prioritizing high contrast, fluid typography, and consistent vertical rhythms.

- **Background:** `#020617` (Deep Indigo/Dark)
- **Primary Accent:** `#a78bfa` (Soft Purple)
- **Secondary Accent:** `#38A6FF` (Vibrant Blue)
- **Accent Highlight:** `#A855F7` (Vibrant Purple)

## 📦 Features

- **Fluid Typography:** Scales seamlessly across all device sizes using `clamp()`.
- **Responsive Layouts:** Mobile-first architecture with safe-area inset support for modern devices.
- **Micro-interactions:** Interactive hover states, orbital animations, and sleek loading sequences.
- **SEO Optimized:** Comprehensive Open Graph (OG) tags, Twitter Cards, and JSON-LD structured data.

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
